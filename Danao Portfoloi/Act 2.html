Table tags:

  • table: The main container for the data grid.
  • thead: Groups header content to allow for separate styling or scrolling.
  • tbody: Groups the primary body content of the table.
  • tfoot: Groups footer/summary content at the bottom.
  • tr: Table Row - Wraps a horizontal set of cells.
  • th: Table Header - Defines a cell as a header (usually bold/centered by default).
  • td: Table Data - Defines a standard cell for your content.
